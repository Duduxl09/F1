ABOUT THIS PROJECT

Este é um projeto inspirado na Fórmula 1, criado por um fã do esporte com o objetivo de recriar a experiência visual de um site no estilo F1.com e transmissões da F1 TV.

O site inclui páginas de pilotos como Oscar Piastri e Lando Norris, além de conteúdo sobre a equipe McLaren e elementos inspirados no mundo da Fórmula 1.

Este projeto foi desenvolvido como forma de aprendizado e evolução na programação, utilizando HTML, CSS e JavaScript para criar uma experiência mais dinâmica e moderna.

Não é um site oficial da Fórmula 1 nem de nenhuma equipe, sendo apenas um projeto de fã.
